## ProxiCraft v1.2.5

### Summary
Fixed mod failing to load on dedicated server hosting panels like CubeCoders AMP.

### Changes
- Added robust path detection that matches how the game itself finds mods
- Mod now checks both AppData location and game install Mods folder

### Installation
1. Download ProxiCraft-1.2.5.zip
2. Extract and copy ProxiCraft folder to 7 Days To Die/Mods/
3. Launch game with EAC disabled

**Multiplayer**: Install on server AND all clients (same version)
